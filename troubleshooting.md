# Guia de Solução de Problemas

Se você encontrar um erro, siga estas etapas para nos ajudar a diagnosticar o problema.

## Verificando o Console do Navegador

1. Abra a página onde o erro ocorre no seu navegador (Google Chrome, Firefox, etc.).
2. Pressione `F12` para abrir as Ferramentas de Desenvolvedor.
3. Clique na aba **Console**.
4. Procure por qualquer mensagem de erro em vermelho.
5. Copie e cole a mensagem de erro completa na nossa conversa.

## Verificando o Log do Servidor

1. Quando você inicia o servidor com `npm run start:server` ou `npm run start:full`, o terminal onde o servidor está rodando exibirá os logs.
2. Realize a ação que causa o erro.
3. Observe o terminal do servidor e procure por qualquer mensagem de erro.
4. Copie e cole a mensagem de erro completa na nossa conversa.
